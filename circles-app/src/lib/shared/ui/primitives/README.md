# Shared UI Primitives

This folder contains low-level, broadly reusable UI building blocks with minimal domain coupling.

## Inclusion criteria
- Used by multiple areas/routes
- Provides structural or interaction primitives (not feature workflows)
- Stable API surface suitable for long-term reuse

## Current primitives
- `tabs/*` — tab system primitives
- `RowFrame.svelte`
- `ActionButton.svelte`
- `Tooltip.svelte`
- `Address.svelte`
- `QrCode.svelte`
